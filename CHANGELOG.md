# Changelog

Todos los cambios notables en este proyecto serán documentados en este archivo.

## [1.0.0] - 2025-02-06

### Agregado
- ✨ Dashboard principal con top 15 criptomonedas
- 📊 Gráficos interactivos con Recharts
- 💼 Gestor de portafolio con P&L
- 🌙 Tema oscuro/claro con next-themes
- 📱 Diseño responsivo completo
- 🔄 Integración con CoinGecko API
- ⚡ Estado global con Zustand
- 🎨 Componentes reutilizables
- 📚 Documentación completa
- 🚀 Build optimizado para Vercel

### Características
- Top 15 criptomonedas con datos en tiempo real
- 4 marcos temporales (1D, 7D, 30D, 1Y)
- Portafolio con cálculo automático de P&L
- Información detallada de cada moneda
- Selector de monedas para análisis
- UI moderna con Tailwind CSS

### Tecnologías
- Next.js 16.1
- React 19.2
- TypeScript 5
- Tailwind CSS 4
- Recharts 3.7
- Zustand 5
- Axios 1.13
